How to make you card work well with our markdown-ish thing:

  1. What is really important and must be exactly (minus capitalisation) as in the given answer can be marked like *something*. This means that if you were to say that *Uno* matches *One* then uno = one, UnO = ONe, Uno != ones, auno != one

  2. Numbers that can be written as a string or as a number can be expressed as -something- for example if Uno is *1* then uno = one, uno = 1, uno != 2, uno != two

  3. Several things that could be included for it to be correct are shown with [thing1, thing2, thing3] for example if A unit for distance is [cm,m,km,miles,centimetres,metres,mm,milimeters,kilometres] then A unit for distance = cm, A unit for distance = metres, A unit for distance != nothing

  4. Things that are not that important can donoted as ^something^, for example if Who is existent is ^the^ government then Who is existent = the government, Who is existent = government, Who is existent != the, Who is existent != anything else

  5. Several things that have to be there are shown as {thing1, thing2, thing3}, for example if What are humans is {mammal, species, intelligent} then What are humans = An intelligent species of the mammal family, What are humans != mammals, What are humans != A mammal species

  6. You can also use several at once! For example you could write [{-1-, cm}, {-1000-, mm}].
